# 我的技术博客

> 使用 VitePress 构建的个人技术博客

## 项目简介

这是一个基于 VitePress 的个人技术博客，主要记录技术笔记、项目分享和工具推荐。

## 技术栈

- **VitePress** - 静态网站生成器
- **Vue 3** - 渐进式前端框架
- **pnpm** - 高效的包管理器

## 项目结构

```
my-blog/
├── docs/                 # 文档目录
│   ├── blog/            # 博客文章
│   ├── .vitepress/      # VitePress 配置
│   └── index.md         # 首页
├── scripts/             # 构建脚本
├── package.json
└── pnpm-lock.yaml
```

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 开发预览

```bash
pnpm docs:dev
```

### 构建生产版本

```bash
pnpm docs:build
```

### 部署

构建完成后，`.vitepress/dist` 目录包含静态文件，可部署到 GitHub Pages、Vercel 等平台。
