## 股价走势：处于近一年低位区间

美团股价从52周高点176.60港元下跌至当前的80.30港元，跌幅已达**54.5%**，股价接近**腰斩**。目前价位接近52周最低点73.60港元，处于明显的低位区间。

从技术面来看：
- 股价已从高点大幅回调，释放了大部分估值压力
- 当前股价在52周区间底部徘徊
- 成交量维持在较高水平，说明市场关注度依然较高

## 业务基本面分析

### 核心业务稳健

美团作为本地生活服务巨头，核心业务包括：

1. **外卖配送** - 市场份额领先，护城河深厚
2. **到店酒旅** - 高毛利业务，盈利能力强劲
3. **美团优选** - 社区团购业务，持续优化
4. **闪购业务** - 即时零售布局，对标京东秒送

### 竞争格局加剧

然而，行业竞争日趋激烈：

- **抖音、快手**：短视频平台凭借流量优势，持续渗透本地生活服务
- **京东**：加码即时零售，"秒送"业务来势汹汹
- **阿里**：饿了么持续投入，淘宝直播切入团购赛道

## 投资观点

### ✅ 支撑因素

1. **估值处于低位**：股价从高点回调超过50%，释放了较大的估值压力
2. **业务护城河深**：外卖和到店业务具有较强的用户粘性
3. **本地生活需求刚性**：消费复苏背景下，弹性可期
4. **回购计划**：公司持续进行股票回购，彰显信心

### ⚠️ 风险因素

1. **竞争加剧**：抖音、京东等平台持续蚕食市场份额
2. **增长放缓**：核心业务增速可能承压
3. **盈利压力**：新业务仍需投入，利润率可能受影响
4. **政策风险**：平台经济监管政策不确定性
5. **宏观经济**：消费复苏力度不及预期
