# 科技股日报 2026年3月20日

今日科技股市场重点关注阿里巴巴、美团、小米三大科技巨头的最新动态。

## 阿里巴巴 (BABA / 09988.HK)

### 财报要点

阿里巴巴将于美东时间5月13日盘前发布2026财年Q4财报：

- **预期营收**：2470.9亿元，同比增长4.50%
- **预期每股收益**：3.83元，同比减少25.93%
- **2026财年全年营收**：10236.7亿元，同比增长3%

若不考虑高鑫零售和银泰的已处置业务的收入，同口径收入同比增长更为显著。阿里在"AI+云"战略布局持续深化，价值重估正在进行中。

### 战略展望

阿里巴巴持续加码AI和云计算投入，战略投入布局未来，核心业务保持稳健增长。

## 美团 (3690.HK)

### 2026年Q1业绩

美团发布2026年第一季度业绩：

- **总收入**：910亿元，同比增长5.6%
- **核心本地商业收入**：641亿元，同比增长0.1%
- **经营利润**：由去年同期的盈利106亿元转为亏损65亿元
- **经营亏损环比大幅收窄**：至20亿元，主要得益于即时配送业务改善

### 市场表现

美团ADR上涨3.3%，显示市场对其中长期发展仍保持信心。尽管竞争加剧拖累利润，但核心业务韧性仍在。

## 小米 (1810.HK)

### 财报动态

小米集团将于5月26日公布2026年第一季度业绩，市场预期关注：

- 智能手机业务表现
- IoT生态增长情况
- 汽车业务最新进展

### 汽车业务

根据乘联分会数据，2026年3月小米汽车批发销量达到21440辆。4月1日小米官方宣布，3月汽车交付量突破2万辆。

小米汽车持续放量，SU7系列市场表现强劲。

### 安全更新

小米本周起陆续为符合更新条件的机型推送2026年6月安卓安全补丁，修复多个安全漏洞。
