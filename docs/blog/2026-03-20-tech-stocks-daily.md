# 科技股日报 2026年3月

今日聚焦三大科技巨头最新动态：阿里巴巴宣布慷慨派息，美团业绩稳中有进，小米汽车交付创新高。

## 阿里巴巴 (BABA/9988.HK)

### 核心新闻
- **派息25亿美元**：阿里巴巴董事会批准派发2026财年年度现金股息，总额约25亿美元
- **AI业务加速**：目标年底ARR（年度经常性收入）突破300亿
- **ESG成果**：2026财年ESG报告显示，自身运营减排量达319.5万吨

### 点评
阿里巴巴在AI商业化方面持续发力，300亿ARR目标显示出对AI业务的信心。慷慨派息回馈股东，ESG表现也彰显企业社会责任。

## 美团 (3690.HK)

### 核心新闻
- **Q1营收增长**：2026年第一季度营收同比增长5.6%至910亿元
- **核心本地商业**：收入同比增长0.1%至641亿元
- **股价表现**：美团ADR上涨3.3%
- **盈利承压**：经营利润由去年同期盈利106亿元转为亏损65亿元

### 点评
美团营收保持增长，但竞争加剧导致利润承压。即时配送业务亏损环比收窄至20亿元，显示出边际改善迹象。
