# 科技股日报 2026年3月20日

今日科技板块热点：阿里巴巴财报即将发布、美团启动大规模春招、小米新品发布会定档。

## 阿里巴巴 (BABA.US / 9988.HK)

### 财报前瞻

阿里巴巴将于 **3月19日美股盘前** 发布2026财年第三季度（截至2025年12月31日）业绩报告。机构预期：

- 营收：2897.3亿元，同比增加3%
- 云业务高歌猛进，传统电商增长乏力

### AI布局加速

- 近期密集发布多款AI大模型
- 钉钉发布AI原生工作平台"悟空"
- 股价表现：3月17日港股盘中涨超3%，报138.2港元

## 美团 (3690.HK)

### 2026春季校招启动

美团正式宣布启动2026届春季校园招聘：

| 类别 | 规模 |
|------|------|
| 2026届补招 | 6000人 |
| 2027届转正实习 | 3000+人 |

**亮点：**
- 大模型等AI技术人才招聘成焦点
- 首招"食杂零售管培生"
- 覆盖技术、产品、商业分析等100+岗位

## 小米 (1810.HK)

### 春季新品发布会

小米2026春季新品发布会定档 **3月19日19:00**，新一代SU7将正式上市。

### 春招同步启动

小米集团2026届春季校园招聘已于3月9日启动，面向海内外应届毕业生。
