# 科技股日报 2026-03-20：阿里AI Agent将亮相 美团本地商业承压 小米机器人新进展

## 阿里巴巴（BABA.US）

**财报要点**：3月19日，阿里发布2025财年Q3财报（截至2025年12月31日止三个月），实现营业收入2848.43亿元，同比增长2%（同口径剔除已处置业务后增）。

**AI布局加速**：据媒体报道，阿里的最快将于本周推出全新的企业级AI Agent应用，由钉钉团队开发，旨在提供更强大的Agent能力，协助企业自动化操作。盘前股价上涨2.7%。

## 美团（3690.HK）

**业绩承压**：3月26日，美团发布2025年Q4及全年业绩，全年实现收入3649亿元，同比增长8%。但受即时零售行业"内卷式"竞争影响，业绩再度变脸。本地商业承压，AI投入加码寻求突围。

**业务进展**：美团单车全面焕新升级，首批新车配备减震系统、舒适座垫，用户反馈良好。

**合规整改**：央视315晚会点名外泌体医美乱象后，美团官方第一时间下架涉事商品。

---

## 小米（1810.HK）

**机器人新进展**：3月27日，雷军发文介绍小米机器人在灵巧手领域的最新进展。该灵巧手采用"全掌触觉手套""仿生结构设计""类人汗腺散热"等多项技术创新。

**产品更新**：小米17系列/REDMI K90系列手机及小米平板8即日起陆续推送澎湃OS 3升级。

**新车发布**：小米再次发布新车，持续布局智能汽车赛道。
