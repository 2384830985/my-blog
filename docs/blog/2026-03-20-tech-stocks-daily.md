# 2026年3月20日 科技股日报：阿里派息25亿美元、AI目标300亿ARR、美团营收增长利润承压

今日港股科技板块整体平稳，三大科技巨头均有重要消息披露：

## 个股分析

### 阿里巴巴 (09988) - 派息+AI双驱动

**核心要点：**

- **Q4财报**：2026财年Q4营收2433.8亿元，同比增长3%
- **派息计划**：董事会批准派发2026财年年度现金股息，总额约25亿美元
- **AI目标**：AI业务目标年底ARR（年度经常性收入）破300亿
- **人事变化**：集团CTO吴泽明进入阿里合伙人委员会，成为第5名成员

**市场解读：**
阿里在保持核心电商业务稳定增长的同时，积极布局AI业务。25亿美元的派息计划体现了对股东回报的重视，AI业务300亿ARR目标显示出管理层对AI商业化的信心。

### 小米集团 (01810) - 汽车交付亮眼

**核心要点：**

- **汽车销量**：2026年3月汽车批发销量达到21440辆
- **财报预期**：将于北京时间5月26日19:30发布第一季度财报
- **系统更新**：本周起陆续推送2026年6月安卓安全补丁

**市场解读：**
小米汽车3月批发销量突破2万辆，显示产能逐步提升。财报发布在即，市场关注点集中在互联网业务与汽车业务的协同效应。

### 美团 (03690) - 营收增长利润承压

**核心要点：**

- **Q1营收**：2026年第一季度总收入同比增长5.6%至910亿元
- **利润下滑**：经营利润由去年同期的盈利106亿元转为亏损65亿元
- **即时配送**：经营亏损环比大幅收窄至20亿元，主要得益于即时配送业务改善

**市场解读：**
美团营收保持增长，但竞争加剧导致利润承压。即时配送业务亏损收窄是唯一亮点，需关注后续业务优化成效。

## 今日观察总结

1. **阿里**：派息+AI双轮驱动，股价有基本面支撑
2. **小米**：汽车销量逐步爬坡，等待财报指引
3. **美团**：营收增长稳健但盈利承压，需观察成本控制
