# 每日科技股分析 2026-03-20

今日科技股市场看点：阿里AI业务迎来商业化拐点，美团深陷外卖烧钱战亏损扩大，小米汽车首盈利但手机业务承压。

## 阿里巴巴 (BABA/9988)

### 核心财报数据

- **Q4营收**: 2433.80亿元，同比增长3%
- **阿里云收入**: 416.26亿元，同比增长38%
- **AI相关产品收入占比**: 首破30%，季度收入89.71亿元
- **AI产品年化ARR**: 突破358亿元

### 关键亮点

1. **AI商业化拐点已至**: 阿里CEO吴泳铭表示，AI已跨越初期培育阶段，正式进入正向规模商业化回报周期
2. **云业务增速创9季新高**: 外部商业化收入增速40%，高于亚马逊AWS，比肩微软Azure
3. **MaaS平台爆发**: 百炼平台客户数量同比增长8倍
4. **股价表现**: 港股一度涨超8%，创2月以来新高

### 风险提示

- 经调整EBITA同比下降84%
- 自由现金流净流出173亿元（去年同期净流入37亿元）
- 对即时零售和AI投入加大导致短期利润承压

## 美团 (3690.HK)

### 核心财报数据

- **2025年营收**: 3649亿元，同比增长8%
- **净亏损**: 234亿元（2024年盈利358亿元）
- **核心本地商业板块**: 从盈利524亿元转为亏损69亿元
- **市场份额**: 约60%（GTV）

### 关键事件

1. **外卖三国杀终结**: 京东份额稳在8%，阿里淘宝闪购DAU增长20%
2. **行业烧钱超6000亿**: 高盛估算三方混战单月最高烧钱250亿元
3. **护城河受考验**: 研发投入260亿元同比增长23.5%，试图用AI建新墙
4. **评级遭下调**: 标普、穆迪下调美团信用评级

### 未来展望

- 京东宣布2026年外卖业务总投入将降低
- 美团推出AI助手"小团"应战
- 阿里明确表示三年不惧亏损，继续加码即时零售

---

## 小米 (1810.HK)

### 核心财报数据

- **2025年营收**: 4573亿元，同比增长25%
- **经调整净利润**: 392亿元，同比增长43.8%
- **汽车业务收入**: 1061亿元，首次突破千亿
- **汽车业务盈利**: 年度经营收益9亿元，首次转正

### 关键亮点

1. **汽车业务爆发**: 全年交付41.1万辆，同比增长200.4%
2. **SU7 Ultra热销**: 20万以上轿车销量第一
3. **2026目标**: 交付55万辆汽车
4. **IoT创新高**: AIoT平台连接设备数达10.79亿台

### 风险提示

- **手机业务承压**: Q1中国出货量暴跌35%，9年来首次跌出前五
- **存储芯片涨价**: 内存成本飙升4倍，性价比机型"出一台亏一台"
- **毛利率下滑**: 手机业务毛利率降至10.9%，Q4单季仅8.3%
