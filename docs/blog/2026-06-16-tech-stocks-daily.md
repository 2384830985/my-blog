# 科技股日报 2026年6月16日

## 阿里巴巴 (9988.HK)

### 核心业绩亮点

**2026财年Q4业绩要点：**
- 收入达2,433.80亿元（人民币），按年增3%
- 经调整净利润降至8,600万元，受战略投入影响
- 阿里云AI业务年化收入（ARR）突破358亿元，三位数增长

### AI业务进展

- AI相关产品收入达89.71亿元，连续第11个季度三位数增长
- AI相关产品收入占阿里云外部商业化收入30%，预计未来一年突破50%
- 百炼MaaS平台客户数量增长八倍
- 模型服务Token消耗规模按季大幅提升
- 6月季度模型与应用服务ARR预计突破100亿元，年末将突破300亿元

### 电商业务

- 即时零售订单规模为去年同期的2.7倍，非餐饮部分达3倍
- 目标：本财年前实现UE（单位经济效益）转正

### 战略投入

- 全财年自由现金流净流出466.1亿元
- 未来两年仍将坚定投资AI
- 服务器GPU几乎无闲置，投资回报确定性强
