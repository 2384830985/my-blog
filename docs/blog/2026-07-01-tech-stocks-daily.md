# 科技股日报 2026-07-01：阿里派息25亿美元、美团Q1盈转亏、小米200亿回购

> 今日聚焦：阿里巴巴宣布25亿美元派息并披露AI商业化目标，美团Q1营收增长但由盈转亏，小米启动200亿港元回购计划

## 阿里巴巴 (BABA/9988.HK)

### 最新动态

**派息与AI目标**
- 阿里巴巴董事会批准派发2026财年年度现金股息，总额约**25亿美元**
- AI业务目标：年底ARR（年度经常性收入）突破**300亿元**
- 相关业务发展获市场关注

**ESG报告发布**
- 阿里巴巴发布2026财年ESG报告
- AI正成为ESG行动的"效率杠杆"和"价值杠杆"
- 自身运营减排量达**319.5万吨**

### 机构观点
市场看好阿里AI商业化前景，派息计划提升股东回报信心。

## 小米 (1810.HK)

### 最新公告

**200亿港元回购计划**
- 小米集团于2026年5月26日披露拟于市场实施**200亿港元**股份购回计划
- 2026年6月18日已与独立经纪商订立协议，执行回购

### 产品与生态

**澎湃 OS 3 升级**
- 新增2026世界杯赛程小部件
- 相册AI修图工具整合
- 采用灰度到全量逐步推送机制

**安卓安全更新**
- 本周起推送2026年6月安卓安全补丁
- 修复多个安全漏洞

### 汽车业务

**3月批发销量**
- 2026年3月小米汽车批发销量达**21,440辆**
- 此前官方宣布3月交付量突破2万辆，但未公布具体数据
