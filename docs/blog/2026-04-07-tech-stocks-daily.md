# 2026年3月科技股月度回顾：AI成核心驱动力

**2026年4月7日** - 回顾刚刚过去的3月，阿里巴巴、美团、小米三家科技巨头各有亮点。AI战略持续深化，成为本轮科技行情的核心主线。

## 阿里巴巴（9988.HK / BABA.US）：AI云业务爆发，财报超预期

### 3月重大事件

**财报发布（3月19日）**
- Q3营收达2897.3亿元，同比+3%
- **阿里云收入增长36%**，成最大亮点
- AI相关产品收入连续**第十个季度**三位数增长

**AI战略加速**
- 3月17日：钉钉发布AI原生工作平台"悟空"
- 股价单日涨超3%，报138.2港元
- 机构关注度提升，AI商业化路径清晰

### 投资观点

阿里云已成第二增长曲线，AI产品收入连续高增长表明商业化进入正循环。电商基本盘稳健，估值仍有修复空间。

## 美团（3690.HK）：春招备战校招季，新车型投放

### 3月动态

**春季校园招聘**
- 2月26日启动，5月31日截止
- 笔试自3月8日起分五场进行
- 覆盖AI研发、产品等热门岗位

**业务创新**
- 3月18日：2026款减震共享单车正式投放
- 新增前减震、手机支架等实用功能
- 率先在北京等城市投放

**财报承压**
- 2025年Q4营收921亿元，同比+4.1%
- 净亏损151.4亿元（上年同期净利润62.2亿元）
- 全年营收约3600亿元量级

### 投资观点

本地生活龙头地位稳固，但即时零售竞争加剧。Q4由盈转亏引发关注，需持续跟踪UE改善节奏。长期看AI与国际化是破局关键。
